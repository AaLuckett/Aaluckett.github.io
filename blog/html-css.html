What's the difference between margin,border and padding in CSS?

Margin is an external element outside the box. It controls the spacing between different elements (boxes) and between the element (box) with the edge of the screen.
Borders are lines drawn on the edge of your element, encasing the element's content, but not ever going outside the element's boundaries.
Padding determines inside spacing, which can control how much the background color is visible around the content.

When to use margin?

  • Change elements position on the page To set the element right or left, up or down on the page, you can play with the margin. To reach the centered position on the page, make sure the width of the page is fixed and set margin value to Auto.
  • Add distance between two elements in the page CSS margins are used to make the elements arrangement on the web page visually appealing by adding spacing between them. This way two images or an image and a text won't densely adjoin each other, distracting the visitor's attention.
